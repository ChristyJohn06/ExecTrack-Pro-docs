# ExecTrack Pro – Field Tracking System for Sales Executives

A modern Field Workforce Management System designed to help organizations monitor, manage, and analyze the activities of field sales executives in real time.

The system enables organizations to track employee locations, manage attendance, assign tasks, monitor customer visits, and generate performance reports through an easy-to-use mobile application and web dashboard.

---

## 📌 Project Overview

ExecTrack Pro is a real-time field employee tracking and management platform developed as an MCA Final Year Project.

The application helps organizations eliminate manual reporting methods by providing a centralized system for:

- Employee Attendance
- Live Location Tracking
- Task Management
- Customer Visit Recording
- Activity Reports
- Performance Analytics

The project is built using Flutter for the mobile application and Django REST Framework for the backend API.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ExecTrackPro.git
```

```bash
cd ExecTrackPro
```

---

### Backend Setup

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Start Server

```bash
python manage.py runserver
```

---

### Flutter Setup

Install Dependencies

```bash
flutter pub get
```

Run Application

```bash
flutter run
```
